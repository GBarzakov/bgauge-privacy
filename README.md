Privacy Policy – BGauge

Last updated: 2025

BGauge does not collect, store or share any personal user data.

The application communicates only with external hardware via Bluetooth in order to display real-time vehicle telemetry. No information from the user or the device is transmitted to any server, stored online, or shared with third parties.

Collected data:
None.

Shared data:
None.

Data stored on the device:
None.

Use of permissions:

Bluetooth – required to communicate with an external ESP32 device for displaying vehicle telemetry.
No other permissions are used.
